# 📚 个人课程表网页版
> 软件技术专业 2025-2026学年第二学期 课程表

## 🎯 项目介绍
这是一个完全还原校园教务系统课表样式的HTML网页，支持电脑/手机自适应打开，可直接部署到GitHub Pages在线访问。

## ✨ 功能特点
- 1:1还原教务系统课表布局、课程颜色、教师/教室信息
- 响应式设计，手机、电脑打开都完美适配
- 纯HTML+CSS实现，无任何依赖，打开即用
- 可自由修改课程信息、配色、布局

## 📂 文件说明
- `666.html`：课程表主页面，直接双击即可在浏览器打开

## 🚀 部署到GitHub Pages（在线访问）
1.  进入仓库的「Settings」（设置）
2.  找到「Pages」选项
3.  「Source」选择「Deploy from a branch」
4.  「Branch」选择「main」，文件夹选择「/ (root)」，点击「Save」
5.  等待1-2分钟，即可通过 `https://你的用户名.github.io/仓库名/` 在线访问你的课表

## 📝 自定义修改
- 修改课程信息：直接编辑`666.html`中对应课程的文字内容
- 修改课程颜色：调整CSS中对应课程类的`background-color`值
- 新增/删除课程：复制/删除对应`td`标签内的课程块即可
